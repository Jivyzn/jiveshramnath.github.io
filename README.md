# Jivesh Ramnath Portfolio

Multi-page static portfolio for GitHub Pages.

## Files
- `index.html`
- `projects.html`
- `achievements.html`
- `research.html`
- `style.css`
- `script.js`

## Deploy
1. Push these files to the GitHub Pages branch or root.
2. Ensure the custom domain is connected in repository settings.
3. Open the domain after DNS propagation.
